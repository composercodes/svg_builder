Odoo SVG Builder

Drawing & Editing Tools

  • Shape Tools: Create basic and complex shapes including Rectangle, Circle, Ellipse, Line, and free-form Paths.
  • L Shape Tool: A specialized L-shaped polyline tool for defining specific areas like booths or stands.
  • Text Tool: Add and edit text directly on the canvas.
  • Selection Tool: Select, move, resize, and rotate objects.
  • Properties Panel: A context-aware panel to edit properties of a selected object, such as fill/stroke color, stroke width, corner radius, font size, and rotation.

Object Manipulation

  • Move: Drag and drop any element on the canvas.
  • Resize: Resize elements using corner and edge handles. Proportional resizing is supported by holding the Shift key.
  • Rotate: Rotate elements using a dedicated rotation handle.
  • Delete: Remove selected elements using the Delete or Backspace key.
  • Constrained Drawing: Hold the Shift key to draw perfect squares, circles, and lines at 45Â° angles.

Canvas & View Controls

  • Pan: Navigate the canvas by holding the Spacebar and dragging, using the middle mouse button, or dragging on an empty area with the select tool.
  • Zoom: Zoom in and out using the mouse wheel. The zoom is centered on the cursor's position for intuitive navigation.
  • Zoom UI: Dedicated toolbar buttons to Zoom In, Zoom Out, and Zoom to Fit the canvas to the view.
  • Grid System: Toggle a visual grid and enable Snap to Grid for precise alignment of objects.
  • Custom Canvas: Adjust the canvas width, height, and background color.
